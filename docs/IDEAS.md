## Near-Future (v0.2–v0.3)

### Member Enhancements
- [ ] Member tags with color coding
- [ ] Member groups/categories (e.g., "Littles", "Protectors")
- [ ] Member search and filter bar
- [ ] Member "status" field (active, dormant, unknown)
- [ ] Soft-archive members without deletion
- [ ] Member-to-member relationships (e.g., "co-fronts often with...")
- [ ] Pinned/favorite members
- [ ] Member "introduction" card (shareable snippet)

### Front Tracking
- [ ] Add note to front entry while it's active
- [ ] Quick-switch front (end current + start new)
- [ ] Front statistics per member (total time, average session)
- [ ] Weekly/monthly front summary
- [ ] Calendar view of front history
- [ ] Scheduled reminders to log front

### Notes
- [ ] Note categories/folders
- [ ] Note search
- [ ] Markdown rendering in notes
- [ ] Pin important notes
- [ ] Note sharing between systems (future)
- [ ] Note versioning/history

### Social / Friend Systems
- [ ] System-to-system invite codes
- [ ] Selective member visibility (choose what to share)
- [ ] View a friend system's current front
- [ ] Public profile mode (shareable link)
- [ ] Private messaging between systems
- [ ] "Fronting buddy" — mutual visibility mode
